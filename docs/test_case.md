# CrownFlow API 测试用例文档

## 1. 测试概述

### 1.1 测试环境
- 后端服务：http://localhost:60000
- 数据库：SQLite（测试用内存数据库）

### 1.2 测试工具
- pytest
- pytest-asyncio
- httpx（异步HTTP客户端）
- factory_boy（测试数据工厂）

### 1.3 测试原则
- 每个测试用例独立运行
- 测试前准备数据，测试后清理数据
- 覆盖正常流程、边界条件、异常场景
